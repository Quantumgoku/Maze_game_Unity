# Maze_game_Unity
Unity Maze Game
 This is my Personal Project of creating a Randomized maze game that is, for each level a random maze will be generated using Backtracking Algorithm for each player.
 
 Considering, Maze being square with spawn Position of a player for 1 vertice, there are other 3 Vertices each having Random chances of being the End position to clear the level. 

The Timer displayed is the time runs from spawning of the Player till the Player reaches the End Position.
